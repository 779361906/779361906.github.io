About Unity

Unity是由Unity Technologies开发的专业游戏引擎。可在Windows 和Mac OS X下开发，可发布游戏至Windows、Mac、Wii、iPhone、Windows phone 8和Android平台。也可以利用Unity web player插件发布网页游戏，支持Mac 和Windows的网页浏览。更多信息请参考百度百科

安装Unity

  1. 下载Unity ，建议到Unity 中文官网下载最新的版本，也可下载我缓存在百度云上的4.5.1版本，地址：http://pan.baidu.com/s/1bnxT48B （ps：百度云的速度会快一些）。

  2. 安装，基本是按默认的设置安装。 Next —> I Agree —> Next —> Install —> Finish。注意：默认安装在C盘，建议安装到D盘。在选择‘install’时将地址C改为D即可。

  3. 启动Unity。Unity启动后会要求你选择激活（Activation），这里先选择本地激活（Manual activation），然后选择标准版或者专业版（专业版是收费的）。启动完成后你看到的界面是英文的，不习惯的话可百度下载汉化补丁，当然还会有不建议使用的破解补丁。

初识Unity

Unity 的操作界面主要分为工程浏览器 (Project Browser)、层级视图 (Hierarchy)、工具栏 (Toolbar)、场景视图 (Scene View)、游戏视图 (Game View)、检视器 (Inspector)和
其他视图 (Views)。具体视图的详细介绍请参考官方文档

创建简单的游戏场景

  1. 创建新工程Project。菜单栏选择File —> New Project 弹出项目初始化资源选项窗口，勾选“Character Controller.unityPackage”、“Standard Assets(Moblile).unityPackage”、“Skybox.UnityPackage”和“Terrain Assets.unityPackage”等4个关键的标准组件，单击“Crate”完成工程文件夹的创建。
  2. 创建场景Scence，场景是一个单独的位置或区域。一个工程里面一般有多个场景。创建过程File —> Save Sence 创建并保存一个场景。（建议在保存路径的时候建立一个Sences文件夹专门存放工程里的场景）。
  3. 为了方便管理工程的文件，建议在工程目录下建立“Models”和“Materials”文件夹用于存放工程的模型文件和材质纹理文件。
